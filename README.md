# Frontend

## How to start developing

**_All scripts should be run from the root directory_**

### 1. Install dependencies

```
npm install
```

### 2. Watch packages

```
npm run dev
```

### 3. Watch app

_Hero (seller) app_

```
npm run dev:hero
```

_Customer (buyer) app_

```
npm run dev:customer
```
