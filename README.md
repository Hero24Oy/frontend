# Frontend

## How to start developing

### 1. Install dependencies

_Run it in the root:_

```
npm install
```

### 2. Watch packages

_We should start watching packages during development, so run it in the root:_

```
npm run dev
```

### 3. Watch app

_Use the following scripts from the apps/\*\*/ directories_

```bash
npm run ios

# or

npm run android
```
